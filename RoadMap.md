# Nota Development RoadMap

This document tracks the development progress of the Nota programming language.

## Phase 1: Core Language Features

- [x] Lexer (Tokenizer)
- [x] Parser (Recursive Descent)
- [x] Abstract Syntax Tree (AST)
- [x] Basic variable declarations (`let`, `mut`)
- [x] Basic data types (`int`, `float`, `bool`, `string`)
- [x] Basic operators (`+`, `-`, `*`, `/`, `%`, `==`, `!=`, `>`, `<`, `>=`, `<=`)
- [ ] Basic control flow (`if`/`else`)
- [x] Comments (`//`, `/* */`)

## Phase 2: Advanced Language Features

- [ ] Functions
- [ ] Advanced operators
- [ ] Match statement
- [ ] Loops (`while`, `for`, `for-each`, `do-while`)
- [ ] Advanced data types (arrays, functions, classes)
- [ ] Type annotations

## Phase 3: Virtual Machine

- [ ] Bytecode definition
- [ ] Bytecode generation from AST
- [ ] Stack-based Virtual Machine
- [ ] Garbage Collection

## Phase 4: Standard Library & Tooling

- [ ] Core library (I/O, Math, etc.)
- [ ] Module system (`import`, `package`)
- [ ] Build and package manager
