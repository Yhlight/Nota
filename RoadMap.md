# Nota Project Roadmap

## Phase 1: Core Language Features

- [x] Lexer
    - [x] Comments (single-line and multi-line)
    - [x] Basic Data Types (int, float, bool)
    - [x] Variable Declaration (let, mut)
    - [x] Basic Operators (+, -, *, /, %)
- [ ] Parser (LL(k) recursive descent)
- [ ] Abstract Syntax Tree (AST)
- [ ] Bytecode Generator
- [ ] Virtual Machine (VM)

## Phase 2: Control Flow

- [ ] If/Else Statements
- [ ] Match Statements
- [ ] While Loops
- [ ] For Loops (C-style)
- [ ] For-Each Loops
- [ ] Do-While Loops

## Phase 3: Functions and Scope

- [ ] Function Declaration and Call
- [ ] Lambda Functions
- [ ] Scope Management

## Phase 4: Advanced Data Types

- [ ] Strings
- [ ] Arrays (dynamic and static)

## Phase 5: Object-Oriented Programming

- [ ] Class Declaration
- [ ] Class Instantiation
- [ ] Member Variables
- [ ] Member Functions (Methods)
- [ ] Constructors

## Phase 6: Modules and Packages

- [ ] Module Import
- [ ] Package Management
