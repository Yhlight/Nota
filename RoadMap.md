# Nota Project Roadmap

This document outlines the development roadmap for the Nota programming language. Each feature will be tracked here, from implementation to completion.

## Phase 1: Project Setup and Core Components

- [x] Project scaffolding (directories, build system)
- [x] Lexer implementation
- [x] Parser implementation (AST)
- [x] Bytecode definition
- [>] Virtual Machine (VM) core
- [ ] Basic REPL

## Phase 2: Language Features

- [ ] Variables (let, mut)
- [ ] Data types (int, float, bool, string)
- [ ] Operators
- [ ] Control flow (if/else, match)
- [ ] Loops (while, for, for-each, do-while)
- [ ] Functions
- [ ] Lambda expressions
- [ ] Classes
- [ ] Modules and packages

## Phase 3: Standard Library and Tooling

- [ ] Core standard library modules
- [ ] File I/O
- [ ] Command-line argument parsing
- [ ] Formatter
- [ ] Linter
- [ ] Debugger

## Phase 4: Optimization and Refinement

- [ ] Performance profiling and optimization
- [ ] Error handling and reporting improvements
- [ ] Code refactoring and cleanup
- [ ] Community feedback and feature requests
