# Nota Project Roadmap

This document tracks the development progress of the Nota language.

## Phase 1: Project Setup and Core Infrastructure

- [x] Initial project scaffolding (CMake, build script, directory structure)
- [ ] Lexer implementation
- [ ] Parser implementation (LL(k) recursive descent)
- [ ] Abstract Syntax Tree (AST) definition
- [ ] Basic bytecode generation
- [ ] Simple Virtual Machine (VM) for executing bytecode

## Phase 2: Language Features

- [ ] Variables (let, mut)
- [ ] Basic data types (int, float, bool)
- [ ] Standard operators
- [ ] Control flow (if/else, while)
- [ ] Functions
- [ ] Advanced data types (string, array)
- [ ] Classes and objects
- [ ] Modules and packages

## Phase 3: Tooling and Ecosystem

- [ ] Standard library
- [ ] File I/O
- [ ] Interactive REPL
- [ ] Debugger support
- [ ] Comprehensive documentation
